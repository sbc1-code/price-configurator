# Price Configurator

Interactive product pricing tool with rule-based validation, real-time cost calculation, and quote generation. Built to replace manual quoting workflows.

## Features

- Multi-product configuration with model/color/beam/housing options
- Rule-based validation (incompatible option combinations flagged)
- Real-time cost calculation with quantity scaling
- Cart system with PDF quote generation
- Light/dark mode toggle

## Stack

JavaScript, HTML/CSS
